📝 Gerenciador de Tarefas (React)

Este é um projeto simples de gerenciamento de tarefas desenvolvido com React, que permite criar, listar e deletar tarefas de forma prática e intuitiva.

🚀 Funcionalidades
✅ Adicionar novas tarefas
📋 Listar tarefas existentes
❌ Remover tarefas
⚡ Atualização instantânea da interface
🧠 Sobre o projeto

Este projeto foi desenvolvido com foco em praticar conceitos fundamentais do React, como:

Componentização
Gerenciamento de estado com useState
Manipulação de eventos
Renderização de listas

⚠️ Observação:
Os dados são mockados (armazenados apenas em memória), ou seja, não há persistência em banco de dados. Ao recarregar a página, as tarefas serão resetadas.

🛠️ Tecnologias utilizadas
React
JavaScript (ES6+)
HTML5
CSS3
